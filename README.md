# @hawiah/drivers [![NPM version](https://img.shields.io/npm/v/@hawiah/drivers.svg?style=flat-square&color=informational)](https://npmjs.com/package/@hawiah/drivers)

Database drivers package for [Hawiah](https://npmjs.com/package/hawiah) framework.

## Installation
```bash
npm install @hawiah/drivers
```

## Available Drivers

- **JSONDriver** - Local JSON file storage
- **YAMLDriver** - Local YAML file storage
- **SQLiteDriver** - SQLite database
- **MongoDriver** - MongoDB database
- **MySQLDriver** - MySQL database
- **PostgreSQLDriver** - PostgreSQL database
- **FirebaseDriver** - Firebase Firestore
- **MemoryDriver** - In-memory storage
